Welcome

Welcome to the website! This is an independent and absolutely free (as well as profit free) project for the Lebanese people and anyone interested in helping bring back Beirut from the horrific explosion that happened on August 4, 2020.

The idea behind this website is to combine all the help and information into one single place: NGOs, independent individuals, organizations, etc... All added here to create one database and portal on how and where one can support the cause. All divided into each respective categories and sorted by their appropriate functions with links to immediate actions or to learn more overall about the entity that is helping Beirut.

There are still more to be added and we'd much appreciate your help if you know any that are missing and need to be added! Please let us know here. (If your organization or name is here and you want to remove it or edit it please let us know as well).

All the code for this website is completely transparent and available to see on Github.
